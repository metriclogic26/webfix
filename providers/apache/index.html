Apache Security & Performance Fixes

Apache deployments rely on mod_headers, mod_rewrite, and sometimes mod_security. HttpFixer documents security headers and connects to CORS patterns when you proxy to a backend.

Because .htaccess and VirtualHost differ per host, live scans prevent guessing which directives apply. SpeedFixer still helps when Apache serves static build output behind TLS termination.
