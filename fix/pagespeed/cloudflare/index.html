Fix PageSpeed Score on Cloudflare

Last updated: April 2026

Cloudflare provides several built-in features that directly improve PageSpeed scores. Enable them in the dashboard.

Get your personalised PageSpeed fix →

Enable compression

Dashboard → Speed → Optimization → Content Optimization: enable Brotli. Cloudflare compresses HTML, CSS, and JS automatically.

Cache rules for static assets

Dashboard → Caching → Cache Rules → Create rule:

When: File extension in {js, css, woff2, png, jpg, webp, svg}
Then: Edge TTL = 1 year, Browser TTL = 1 year
      Cache-Control header: public, max-age=31536000, immutable

Enable Early Hints

Dashboard → Speed → Optimization → Early Hints: On. Cloudflare sends HTTP 103 responses for preloaded resources, allowing browsers to start fetching before the full response arrives.

Enable HTTP/2 and HTTP/3

Dashboard → Network: enable HTTP/2 and HTTP/3 (QUIC). HTTP/3 reduces connection overhead, especially on mobile.

Polish — automatic image optimisation

Dashboard → Speed → Optimization → Polish: Lossless or Lossy. Compresses images on the fly. Lossy provides larger file size reductions.
